2. 5 registered users:

    2. mike

3. supports registration and login.
4. no server.
5. great design.

## How To Run
1. clone repo
2. install dependencies using `npm i`
3. run site using `npm start`
4. open `http://localhost:3000/` with a browser

### Dependencies
this site uses:
- react
- react router
- bootstrap
